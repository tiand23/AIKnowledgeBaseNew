# Security Policy

## Supported Versions

This project is currently in an early OSS phase.

| Version | Supported |
|---------|-----------|
| main (latest) | Yes |
| older tags/releases | No |

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Report through one of the following private channels:
- Email: `security@example.com` (replace before public release)
- Alternative contact: project maintainers

Please include:
- Affected component/path
- Reproduction steps
- Impact assessment
- Suggested mitigation (optional)

## Response SLA (Target)

- Initial acknowledgement: within 72 hours
- Triage result: within 7 business days
- Fix plan: announced after impact assessment

## Disclosure Policy

- Coordinated disclosure is preferred.
- Public disclosure should happen after a fix or mitigation is available.
